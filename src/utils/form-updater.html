Story Creator Form Field Updater

Note: This tool helps fix issues with form fields not updating properly in the Story Creator.

Instructions

  1. Open the Story Creator page in your browser
  2. Open the browser's developer console (F12 or right-click > Inspect > Console)
  3. Copy the script below
  4. Paste it into the console and press Enter
  5. Use the updateStoryFields() function as described below

The Script

How to Use

After pasting the script in the console, you can use the following function to update form fields:

updateStoryFields(
  "Your Story Title", 
  "Your Story Description", 
  ["Acceptance Criterion 1", "Acceptance Criterion 2"]
);

For example:

updateStoryFields(
  "Implement User Login", 
  "As a user, I want to log in to the system so that I can access my account.", 
  [
    "Given I am on the login page, when I enter valid credentials, then I should be logged in successfully.",
    "Given I am on the login page, when I enter invalid credentials, then I should see an error message."
  ]
);

Automatic Monitoring

The script also automatically monitors for AI-generated content and will attempt to update the form fields when it detects a response from the AI service.
